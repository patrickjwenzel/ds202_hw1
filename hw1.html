  7. Install git.
    • If you are on Windows:
      1. Install Git for Windows. In the coming weeks we will use the Git Bash (but not the Git GUI).
      2. After installation, set up Rstudio to use Git bash as the shell for the Terminal: Open RStudio, click Tools–Global Options–Terminal; under Shells, select Git Bash; click OK.
      3. In the main menu of RStudio: click Tools–Terminal–New Terminal. If you have a $ before your prompt, then you should be good to go.
    • If you are on Linux or macOS, Git has probably been installed by default, so you do not need to install anything. Just click the Terminal tab (above the R Console).
